Project Overview

Bitzapp is a hackathon-ready MVP that brings Bitcoin wallet functionality directly into WhatsApp. Users can send, receive, and manage Bitcoin through simple text commands, with support for both custodial and non-custodial wallets, Lightning Network payments, and AI-powered assistance.

Built specifically for the Nigerian market, Bitzapp integrates with local payment providers like Bitnob and MavaPay to enable seamless Bitcoin-to-Naira conversions, bill payments, and bank withdrawals.

Key Technologies:
• Django + Django REST Framework
• SQLite (hackathon) / PostgreSQL (production)
• Bitnob API (Bitcoin/Naira conversion)
• MavaPay API (Bill payments)
• Gemini AI (Chatbot)
• WhatsApp Business API

Wallet Types

Non-Custodial Wallets

Maximum security where users control their private keys through 12-word seed phrases.

Advantages:
• You control your private keys
• True decentralization
• No single point of failure
• Maximum security
Considerations:
• Must safely store seed phrase
• Requires seed phrase for transactions
• No recovery if seed phrase is lost

Custodial Wallets

Easy-to-use wallets where Bitzapp manages private keys for convenience.

Advantages:
• Easy to use
• No seed phrase needed
• Simple transactions
• Good for beginners
Considerations:
• We control your private keys
• Centralized control
• Less decentralized

Quick Start Guide

Step 1: Access Bitzapp

Click any WhatsApp button on the landing page or visit: https://wa.me/message/X3JCWOTQ5IKEF1

Step 2: Create Your Wallet

Command:
/create
Response: Creates non-custodial wallet with seed phrase

Step 3: Check Your Balance

Command:
/balance
Response: Shows Bitcoin balance and Naira equivalent

Step 4: Start Using Bitcoin

Deposit Naira:
/deposit 50000
Create Lightning Invoice:
/lightning 1000
Pay Bills:
/paybill mtn 1000 08012345678

Wallet Management Commands

/create

Creates a new non-custodial Bitcoin wallet with a 12-word seed phrase.

Usage:
/create
Response: Generates seed phrase and Bitcoin address
Warning:
Save your seed phrase safely! We cannot recover it if lost.

/import

Imports an existing wallet using a 12-word seed phrase.

Usage:
/import word1 word2 word3 ... word12
Response: Imports wallet and shows Bitcoin address

/wallet

Shows information about wallet types and their differences.

Usage:
/wallet
Response: Explains custodial vs non-custodial wallets

/balance

Shows your current Bitcoin balance and Naira equivalent.

Usage:
/balance
Response: Shows BTC amount, NGN equivalent, and wallet type

Lightning Network Commands

/lightning

Creates a Lightning Network invoice for instant Bitcoin payments.

Usage:
/lightning <amount_sats> [description]
Examples:
/lightning 1000
/lightning 5000 Coffee payment
Lightning Benefits:
Instant payments, micro-fees, scalable network

/lightningpay

Pays a Lightning Network invoice using a payment request.

Usage:
/lightningpay <payment_request> [description]
Example:
/lightningpay lnbc1000u1p... Coffee payment

/lightningstatus

Checks the status of a Lightning invoice.

Usage:
/lightningstatus <invoice_id>
Response: Shows payment status and details

/lightninghistory

Shows your Lightning Network transaction history.

Usage:
/lightninghistory
Response: Lists recent Lightning invoices and payments

Bitcoin Operations Commands

/send

Sends Bitcoin to another address.

Custodial Usage:
/send <amount> <address>
Non-Custodial Usage:
/send <amount> <address> <seed_phrase>
Example:
/send 0.001 bc1qxy2kgdygjrsqtzq2n0yrf2493p83kkfjhx0wlh

/receive

Gets your Bitcoin address for receiving payments.

Usage:
/receive
Response: Shows your Bitcoin address

/deposit

Creates a Naira deposit request to convert to Bitcoin.

Usage:
/deposit <amount_ngn>
Example:
/deposit 50000
Response: Shows Bitcoin equivalent and deposit instructions

/withdraw

Withdraws Bitcoin to a Nigerian bank account.

Usage:
/withdraw <amount_btc> <account_number> <bank_name> <account_name>
Example:
/withdraw 0.001 1234567890 Access Bank John Doe

Bill Payment Commands

/paybill

Pays Nigerian bills using Bitcoin.

Usage:
/paybill <provider> <amount> <account>
Examples:
/paybill mtn 1000 08012345678
/paybill airtel 2000 08098765432
/paybill electricity 5000 12345678901
Supported Providers:
MTN, Airtel, Electricity, DSTV, School Fees

/bills

Lists available bill payment providers.

Usage:
/bills
Response: Shows all supported bill providers

Technical Architecture

Django Application Structure

Project Structure:
bitzapp/
├── core/ # Core models and views
│ ├── models.py # User, Wallet, Transaction models
│ ├── views.py # WhatsApp webhook handler
│ └── command_handlers.py # Command processing logic
├── wallet/ # Bitcoin wallet services
│ ├── models.py # Wallet-specific models
│ └── services.py # Bitcoin operations
├── payments/ # Payment processing
│ ├── models.py # Payment models
│ └── services.py # Bitnob/MavaPay integration
├── chatbot/ # AI chatbot
│ ├── models.py # Chat models
│ └── services.py # Gemini AI integration
└── tests/ # Test scripts

Database Models

Core Models:
• BitzappUser - Extended user with phone number
• BitcoinWallet - Bitcoin wallet with address and balance
• Transaction - All Bitcoin transactions
• ExchangeRate - BTC/NGN exchange rates
• BillProvider - Bill payment providers
Payment Models:
• BillPayment - Bill payment transactions
• NairaDeposit - Naira to Bitcoin deposits
• NairaWithdrawal - Bitcoin to Naira withdrawals
• LightningInvoice - Lightning Network invoices
• LightningPayment - Lightning Network payments

Service Layer

Services:
• BitcoinWalletService - Wallet operations
• PaymentService - Payment processing
• AIChatbotService - AI integration
Key Methods:
• create_wallet_for_user() - Wallet creation
• send_bitcoin() - Bitcoin transactions
• create_lightning_invoice() - Lightning invoices
• pay_bill() - Bill payments
• process_ai_chat() - AI responses

API Integration

Bitnob API

Primary Bitcoin/Naira conversion and Lightning Network provider.

Endpoints Used:
• /api/v1/rates/btc - Get BTC/NGN exchange rate
• /api/v1/deposits - Create Naira deposits
• /api/v1/withdrawals - Process Naira withdrawals
• /api/v1/lightning/invoices - Create Lightning invoices
• /api/v1/lightning/payments - Process Lightning payments
Environment Variable:
BITNOB_API_KEY=your_bitnob_api_key

MavaPay API

Bill payment processing for Nigerian services.

Endpoints Used:
• /api/v1/bills/pay - Process bill payments
Supported Providers:
• MTN, Airtel, Electricity, DSTV, School Fees
Environment Variable:
MAVAPAY_API_KEY=your_mavapay_api_key

Gemini AI API

AI-powered chatbot for Bitcoin education and assistance.

Features:
• Bitcoin education and explanations
• Command help and guidance
• Market insights and trends
• Natural language processing
Environment Variable:
GEMINI_API_KEY=your_gemini_api_key

WhatsApp Business API

Message handling and webhook integration.

Webhook URL:

Deployment Guide

Environment Setup

Required Environment Variables:
# WhatsApp Business API
WHATSAPP_ACCESS_TOKEN=your_whatsapp_token
WHATSAPP_PHONE_NUMBER_ID=your_phone_number_id
WHATSAPP_VERIFY_TOKEN=your_verify_token
# Bitnob API
BITNOB_API_KEY=your_bitnob_api_key
BITNOB_BASE_URL=https://api.bitnob.com
# MavaPay API
MAVAPAY_API_KEY=your_mavapay_api_key
MAVAPAY_BASE_URL=https://api.mavapay.co
# Gemini API
GEMINI_API_KEY=your_gemini_api_key

Database Migration

Commands:
python manage.py makemigrations
python manage.py migrate
python manage.py setup_sample_data

Production Deployment

Recommended Stack:
• PostgreSQL database
• Redis for caching
• Celery for background tasks
• Gunicorn for production server
• Nginx for reverse proxy
• SSL certificates for HTTPS
