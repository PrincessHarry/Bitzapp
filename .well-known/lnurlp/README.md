# LNURL Configuration for Bitzapp

This directory contains the LNURL configuration for Lightning addresses on the `bitzapp-i3i3.onrender.com` domain.

## How it works:

1. When someone sends to `username@bitzapp-i3i3.onrender.com`
2. The Lightning wallet queries `https://bitzapp-i3i3.onrender.com/.well-known/lnurlp/username`
3. Our Django app returns the LNURL configuration
4. The sender's wallet can then send the payment

## File Structure:
```
.well-known/
└── lnurlp/
    └── {username}  # Dynamic files generated by Django
```

## Example:
- Lightning address: `alice@bitzapp-i3i3.onrender.com`
- LNURL endpoint: `https://bitzapp-i3i3.onrender.com/.well-known/lnurlp/alice`
- Returns: LNURL configuration JSON for Alice's wallet
